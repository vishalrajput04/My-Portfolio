## 📝 How to Use
1. Clone this repository:
   ```sh
   git clone https://github.com/your-username/your-portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd your-portfolio
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
